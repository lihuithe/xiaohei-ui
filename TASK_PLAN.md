# 小黑助手 - UI 组件库任务规划

## 项目目标

打造一个高质量、易于使用的 shadcn-vue 风格的 Electron + Vue 3 UI 组件库，包含完整的桌面应用框架、组件示例、文档、开发工具等。

## 当前状态分析

### 已有的内容
- ✅ shadcn-vue 完整组件库（40+ 组件）
- ✅ Electron + Vue 3 + TypeScript + Vite 框架
- ✅ macOS 毛玻璃透明窗口
- ✅ 侧边栏布局
- ✅ 系统托盘功能
- ✅ Tailwind CSS v4 主题系统

### 缺失的内容
- ❌ 组件示例和演示页面
- ❌ 主题切换（浅色/深色）
- ❌ 组件使用文档
- ❌ 开发工具和脚手架
- ❌ 更多实用布局模板
- ❌ 测试覆盖
- ❌ CI/CD 自动化
- ❌ 国际化支持
- ❌ 动画和过渡效果
- ❌ 代码规范和格式化

## 分支管理策略

- `main` - 主分支，稳定版本
- `agent-1/xxx` - Agent 1 开发分支
- `agent-2/xxx` - Agent 2 开发分支
- `feature/xxx` - 功能分支

## 工作流程

1. Agent 创建自己的功能分支
2. 在分支上完成任务
3. 完成后提交和推送到远程
4. 我（主 Agent）负责代码审查和合并
5. 有 Bug 则反馈给对应 Agent 修复

## 验收标准

- 代码符合 TypeScript 规范
- 没有 ESLint 错误
- 组件功能完整可用
- 跨平台兼容性良好（macOS/Windows/Linux）
- 文档完整清晰
