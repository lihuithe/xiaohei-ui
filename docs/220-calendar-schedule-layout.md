# 220 日历/日程布局（Calendar & Schedule Layout）

## 功能名称

日历/日程布局 — 提供月视图日历、周日程视图、时间线日程表三种可复用骨架。

## 功能背景/动机

现有脚手架已有 `ui/calendar/` 和 `ui/range-calendar/` 组件族，提供了完整的日历日期选择器。但日历作为**页面级布局**时，通常需要更丰富的展示：事件列表、日程安排、周视图的时间网格、月视图的日期格子等。`Calendar` 组件本身是一个日期选择控件，不是日程展示布局。开发者在实现日程管理、预约系统、计划看板时，需要自行设计日历的页面级结构。

本功能提供三种日历/日程的页面级布局模板，与现有的 `Calendar` 组件衔接，但不重复实现日期计算逻辑。

## 功能描述

包含以下三种日历/日程布局变体：

| 变体 | 说明 | 适用场景 |
|------|------|----------|
| `MonthCalendarLayout` | 月视图：日期格子矩阵 + 每日事件摘要 + 侧边详情面板 | 日程概览、活动计划、任务截止日期 |
| `WeekScheduleLayout` | 周日程视图：时间轴（0-24时）+ 事件块拖拽占位 | 周计划、会议安排、工时记录 |
| `TimelineScheduleLayout` | 时间线日程表：水平时间轴 + 事件卡片 | 项目里程碑、发布计划、活动排期 |

每种布局均内建：
- 日期导航控制（上一月/周、下一月/周、今天、日期选择器）
- 事件/日程的展示槽位（接受任意卡片组件）
- 空日期状态（`Empty` 组件："本日无安排"）
- 与 `ui/calendar/` 的衔接（日期选择器 popover）
- 暗色/亮色主题适配

## 目标用户

- 需要实现日程管理、计划安排、时间线展示的开发者
- 需要在仪表盘内嵌入小型日历/日程预览的开发者

### 涉及的技术点

- 使用现有 `ui/calendar/` 的 `Calendar` 作为日期选择 popover
- 使用 CSS Grid 构建月视图的 7xN 日期矩阵
- 使用 CSS Grid 或 `flex` + 固定时间刻度构建周视图的时间轴
- 使用 `ScrollArea` 实现周视图和时间线视图的独立滚动
- 事件块使用绝对定位或 Grid 行/列跨度实现跨时段/跨日期展示
- 日期导航使用 `date-fns` 或原生 `Date` 进行计算（布局模板不绑定具体库）

### 与现有 layouts/ 的衔接

- 新增 `src/components/layouts/MonthCalendarLayout.vue`
- 新增 `src/components/layouts/WeekScheduleLayout.vue`

**Props 设计（以 WeekScheduleLayout 为例）：**

```ts
interface ScheduleEvent {
  id: string
  title: string
  start: Date
  end: Date
  color?: string           // 事件块颜色
  description?: string
}

interface WeekScheduleLayoutProps {
  events: ScheduleEvent[]
  startHour?: number       // 时间轴起始小时，默认 8
  endHour?: number         // 时间轴结束小时，默认 18
  hourHeight?: number      // 每小时高度(px)，默认 60
  showWeekend?: boolean    // 是否显示周末，默认 true
}

// Slot 约定
// #event="{ event }"  — 自定义事件块渲染
// #empty-day          — 空日期状态
// #header-extra       — 头部额外操作（如新建日程按钮）
```

## 验收标准

- [ ] 三种布局均能在 `ComponentPlayground` 中独立预览
- [ ] `MonthCalendarLayout` 的日期格子支持通过 `#day-content` slot 自定义内部事件展示
- [ ] `WeekScheduleLayout` 的时间轴支持横向滚动查看整周，时间刻度列保持固定
- [ ] `TimelineScheduleLayout` 的事件块支持跨越多个月份，宽度按比例计算
- [ ] 点击日期/事件触发相应的事件（`@date-click`、`@event-click`）
- [ ] 空日期或空时间槽展示 `Empty` 占位（"本日无安排"）
- [ ] 当前日期/当前时间线有视觉高亮（如红色竖线或边框）
- [ ] 暗色模式下日期格子的边框和事件块颜色自动适配
- [ ] 所有日期格式通过 props 传入已格式化字符串，不绑定具体日期库

## 优先级

P1 — 日历/日程布局是时间管理类桌面应用的核心，与已有的 `Calendar` 组件形成页面级补充。
