# 219 预览/阅读器布局（Preview & Reader Layout）

## 功能名称

预览/阅读器布局 — 提供文档阅读器、媒体预览器、图片画廊三种沉浸式浏览骨架。

## 功能背景/动机

桌面应用中经常需要让用户专注于内容的阅读或预览：查看文档、浏览图片集、预览媒体文件。现有脚手架已有 `ScrollArea`、`AspectRatio`、`Carousel` 等组件，但缺少**面向阅读和预览场景的页面级布局模板**。开发者在实现阅读器时，需要自行处理内容区的最大宽度限制（保证可读性）、导航控制（上一页/下一页）、缩略图/目录侧边栏、以及全屏沉浸模式等通用结构。

本功能提供三种专注于"内容消费"的布局模板，与"内容生产"类的编辑器布局（216）形成互补。

## 功能描述

包含以下三种预览/阅读器布局变体：

| 变体 | 说明 | 适用场景 |
|------|------|----------|
| `DocumentReaderLayout` | 文档阅读器：目录侧边栏 + 主内容区（最大宽度限制）+ 阅读进度 | 长文档、帮助中心、电子书 |
| `MediaPreviewLayout` | 媒体预览器：主预览区 + 底部/侧边缩略图栏 + 播放控制 | 视频预览、音频预览、幻灯片 |
| `ImageGalleryLayout` | 图片画廊：网格缩略图 + 灯箱/全屏查看 + 键盘导航 | 相册浏览、设计稿审阅、截图库 |

每种布局均内建：
- 内容区的最大宽度限制（保证可读性，默认 768px 居中）
- 导航控制（上一项/下一项、页码/计数器）
- 缩放控制（放大/缩小/适应屏幕/原始尺寸）
- 全屏/沉浸模式切换
- 与 `ScrollArea` 的衔接（长内容独立滚动）

## 目标用户

- 需要实现文档阅读、媒体预览、图片浏览功能的开发者
- 希望提供沉浸式、无干扰内容消费体验的开发者

### 涉及的技术点

- 使用 `max-w-3xl mx-auto` 限制文档阅读区的最大宽度（约 768px）
- 使用 `ScrollArea` 包裹目录侧边栏和文档内容区，实现独立滚动
- 使用 `ResizablePanelGroup` 实现文档阅读器的目录面板拖拽调宽
- 使用现有 `Carousel` 组件作为媒体预览器的缩略图栏底层
- 使用 `AspectRatio` 组件保持媒体预览区的比例
- 灯箱模式使用固定定位（`fixed inset-0`）覆盖全屏

### 与现有 layouts/ 的衔接

- 新增 `src/components/layouts/DocumentReaderLayout.vue`
- 新增 `src/components/layouts/MediaPreviewLayout.vue`
- 新增 `src/components/layouts/ImageGalleryLayout.vue`
- 在 `src/components/layouts/index.ts` 中导出

**Props 设计（以 DocumentReaderLayout 为例）：**

```ts
interface TocItem {
  id: string
  title: string
  level: number            // 标题层级，用于缩进
}

interface DocumentReaderLayoutProps {
  title: string
  toc?: TocItem[]          // 目录数据
  showToc?: boolean        // 是否显示目录，默认 true
  maxContentWidth?: number // 内容区最大宽度，默认 768
  showProgress?: boolean   // 是否显示阅读进度，默认 true
}

// Slot 约定
// #toc-header    — 目录面板顶部（如文档元信息）
// #content       — 文档主内容（必填）
# #toolbar-right — 顶部工具栏右侧操作
```

## 验收标准

- [ ] 三种布局均能在 `ComponentPlayground` 中独立预览
- [ ] `DocumentReaderLayout` 的内容区在宽屏下居中，最大宽度不超过 768px，两侧留白舒适
- [ ] `DocumentReaderLayout` 的目录支持点击跳转，当前阅读章节高亮显示
- [ ] `MediaPreviewLayout` 的主预览区使用 `AspectRatio` 保持比例，窗口变化时自适应
- [ ] `ImageGalleryLayout` 的灯箱模式支持键盘左右方向键切换图片，ESC 退出
- [ ] 灯箱模式下图片支持缩放（鼠标滚轮或按钮），双击恢复原始尺寸
- [ ] 阅读进度根据内容区滚动位置实时计算，误差小于 5%
- [ ] 全屏模式下隐藏系统标题栏/浏览器 chrome，最大化内容展示区域

## 优先级

P1 — 阅读/预览布局是内容消费型桌面应用的核心，与编辑器布局形成完整闭环。
