# 203 时间线布局模板（Timeline Layout）

## 功能名称

时间线布局模板 — 提供垂直时间线（左右交替 / 左对齐）、水平时间线两种可复用骨架。

## 功能背景/动机

时间线是展示事件序列、操作历史、版本变更、项目里程碑的理想布局。现有脚手架已有丰富的原子组件（`Card`、`Badge`、`Avatar`、`Empty`、`Separator`），但缺少**页面级的时间线编排模板**。开发者在实现时间线时，需要自行处理中轴线绘制、节点对齐、内容卡片位置计算等重复工作。

本功能提供纯布局骨架，不绑定具体业务数据格式，让开发者只需传入事件数组即可渲染出一致风格的时间线。

## 功能描述

包含以下三种时间线布局变体：

| 变体 | 说明 | 适用场景 |
|------|------|----------|
| `TimelineVerticalAlternating` | 垂直时间线，内容左右交替排列 | 产品迭代历史、公司发展历程 |
| `TimelineVerticalLeft` | 垂直时间线，所有内容左对齐，轴线在左侧 | 操作日志、审计记录、消息通知 |
| `TimelineHorizontal` | 水平时间线，节点横向排列 | 项目里程碑、步骤概览、注册流程 |

每种布局均内建：
- 节点标记（圆点 / 图标 / 数字 / 自定义）
- 连接线（自动计算并绘制轴线）
- 时间戳展示位置（节点上方/下方/卡片内部）
- 与 `Card`、`Badge` 的无缝嵌套
- 支持"加载更多"占位和空状态（`Empty` 组件）

## 目标用户

- 需要展示事件流、历史记录、版本变更的开发者
- 需要在仪表盘或详情页内嵌入时间线片段的开发者

### 涉及的技术点

- 使用 CSS `::before` / `::after` 伪元素绘制轴线，避免引入额外 DOM
- 使用 `grid` 布局实现左右交替的精准对齐
- 使用现有 `Card`、`Badge`、`Avatar` 作为时间线节点内容容器
- 使用现有 `Empty` 组件处理无事件状态
- 节点状态通过 `variant` 区分：`default`（已完成）、`active`（当前）、`upcoming`（待发生）

### 与现有 layouts/ 的衔接

- 新增 `src/components/layouts/TimelineVerticalAlternating.vue`
- 新增 `src/components/layouts/TimelineVerticalLeft.vue`
- 新增 `src/components/layouts/TimelineHorizontal.vue`
- 在 `src/components/layouts/index.ts` 中导出
- 可作为 `DetailStackedLayout` 中某个 Tab 的内容嵌入

### 需要新增/修改的文件

```
desktop-app/src/components/layouts/
  ├── TimelineVerticalAlternating.vue   # 左右交替垂直时间线
  ├── TimelineVerticalLeft.vue          # 左对齐垂直时间线
  ├── TimelineHorizontal.vue            # 水平时间线
  └── index.ts                          # 追加导出
```

**Props 设计（以 TimelineVerticalLeft 为例）：**

```ts
interface TimelineItem {
  id: string
  title: string
  description?: string
  timestamp?: string | Date
  variant?: 'default' | 'active' | 'upcoming'
  icon?: Component                    // 自定义节点图标
  dotColor?: string                   // 节点颜色，覆盖 variant
}

interface TimelineVerticalLeftProps {
  items: TimelineItem[]
  reverse?: boolean                   // 是否倒序（最新在上）
  loading?: boolean                   // 是否加载中（展示 Skeleton）
}
```

## 验收标准

- [ ] 三种时间线布局均能在 `ComponentPlayground` 中独立预览
- [ ] 垂直时间线的轴线在内容高度变化时自动延伸，无断线
- [ ] 左右交替布局在窗口宽度 < 768px 时自动降级为左对齐布局
- [ ] 水平时间线在节点数溢出时支持水平滚动，且当前激活节点自动滚动到可视区（可选）
- [ ] 支持通过 `icon` 插槽或 prop 自定义节点图标（默认圆点）
- [ ] 空数据时展示 `Empty` 占位，加载时展示 `Skeleton` 占位
- [ ] 时间戳支持国际化日期格式（通过 `vue-i18n` 或传入已格式化字符串）

## 优先级

P1 — 时间线是详情页和日志页的重要补充，使用频率低于表单和看板，但对脚手架完整性很重要。
