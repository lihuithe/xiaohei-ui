# 227 地图/可视化布局占位（Map & Visualization Layout）

## 功能名称

地图/可视化布局占位 — 提供地图视图、数据可视化仪表盘、关系图谱三种可复用骨架。

## 功能背景/动机

桌面应用中，数据可视化是展示复杂信息的有效手段：地理分布、网络拓扑、组织架构、数据关联等。现有脚手架已有 `ChartContainer`（`ui/chart/`）组件用于常规图表，但缺少**面向地图和复杂可视化场景的页面级布局模板**。这些场景有其独特的布局需求：地图需要全屏或大幅面展示、可视化仪表盘需要图表矩阵、关系图谱需要无边界的画布空间。

本功能提供纯粹的 UI 布局骨架，不绑定任何具体的地图库（如 Leaflet、Mapbox）或可视化库（如 D3、ECharts），只提供容器布局和控件编排。

## 功能描述

包含以下三种可视化布局变体：

| 变体 | 说明 | 适用场景 |
|------|------|----------|
| `MapViewLayout` | 地图视图：全屏地图容器 + 浮动控件（缩放/图层/搜索）+ 底部信息卡 | 地理信息系统、位置管理、配送追踪 |
| `VisualizationDashboardLayout` | 可视化仪表盘：图表矩阵 + 控制面板 + 图例区 | 数据分析、BI 报表、监控大屏 |
| `GraphNetworkLayout` | 关系图谱：无边界画布 + 悬浮节点详情 + 缩放/平移控件 | 组织架构、知识图谱、网络拓扑 |

每种布局均内建：
- 渲染容器的自适应尺寸（占满可用空间）
- 浮动控件的面板（可收起/展开）
- 图例/说明区
- 加载状态（地图瓦片/数据加载中的骨架占位）
- 空数据状态（无标记点、无节点时的 `Empty` 占位）

## 目标用户

- 需要集成地图、图表矩阵、关系图谱的开发者
- 需要大数据可视化展示界面的开发者

### 涉及的技术点

- 使用 `flex` + `overflow-hidden` 实现渲染容器的自适应占满
- 使用 `absolute` 定位实现浮动控件覆盖在渲染层之上
- 使用 `Card` 组件构建信息卡片和节点详情面板
- 使用 `ui/skeleton/Skeleton.vue` 作为加载占位（地图的灰色方块矩阵）
- 使用 `ui/empty/` 处理无数据状态
- 使用 `ButtonGroup` 实现缩放控制按钮组

### 与现有 layouts/ 的衔接

- 新增 `src/components/layouts/MapViewLayout.vue`
- 新增 `src/components/layouts/VisualizationDashboardLayout.vue`

- 在 `src/components/layouts/index.ts` 中导出

- `GraphNetworkLayout` 可与 `ClassicCanvasLayout`（224）组合（图谱作为画布内容）

**Props 设计（以 MapViewLayout 为例）：**

```ts
interface MapViewLayoutProps {
  showSearch?: boolean       // 是否显示搜索框，默认 true
  showZoomControls?: boolean // 是否显示缩放控件，默认 true
  showLayerSwitcher?: boolean// 是否显示图层切换，默认 true
  showInfoCard?: boolean     // 是否显示底部信息卡，默认 true
  loading?: boolean
}

// Slot 约定
// #map           — 地图渲染区域（必填）
// #search        — 自定义搜索控件
// #controls      — 自定义浮动控件
// #info-card     — 底部信息卡内容
// #legend        — 图例内容
```

## 验收标准

- [ ] 三种布局均能在 `ComponentPlayground` 中独立预览
- [ ] `MapViewLayout` 的地图渲染区域占满所有可用空间，无固定高度限制
- [ ] 浮动控件（搜索、缩放按钮）在地图缩放/平移时保持固定位置，不随地图移动
- [ ] `VisualizationDashboardLayout` 的图表网格在窗口变化时自动重排，无横向滚动
- [ ] `GraphNetworkLayout` 的节点详情面板在点击节点后 150ms 内弹出，点击空白处关闭
- [ ] 加载状态时，地图区域展示网格状 `Skeleton` 占位，图表区展示卡片骨架屏
- [ ] 空数据时展示 `Empty` 占位（地图图标 + "暂无位置数据"）
- [ ] 所有布局不绑定具体渲染库，`#map` / `#chart` / `#graph` slot 可放置任意组件

## 优先级

P1 — 可视化布局是数据密集型桌面应用的核心展示方式。
