# 232 可展开列表布局（Expandable List Layout）

## 功能名称

可展开列表布局 — 提供主从列表、嵌套树形列表、分步展开列表三种可复用骨架。

## 功能背景/动机

桌面应用中，列表数据往往具有层级或关联关系：邮件列表展开显示正文、订单列表展开显示商品明细、任务列表展开显示子任务。现有脚手架已有 `ui/accordion/`（手风琴）和 `ui/collapsible/`（可折叠）组件，但它们是独立的折叠区块，缺少**面向列表场景的、可展开的行级布局模板**。开发者在实现展开列表时，需要自行处理展开图标旋转动画、展开内容的高度过渡、以及展开状态的记忆。

本功能提供三种专门的展开列表布局，统一展开/收起的交互规范和动画行为。

## 功能描述

包含以下三种可展开列表布局变体：

| 变体 | 说明 | 适用场景 |
|------|------|----------|
| `MasterDetailListLayout` | 主从列表：点击主行展开详情面板，详情在下方或右侧 | 邮件列表、订单列表、审批流 |
| `NestedListLayout` | 嵌套列表：每行可展开显示子列表，支持无限层级 | 文件夹树、分类目录、组织架构 |
| `StepExpandListLayout` | 分步展开列表：列表项按步骤顺序展开，完成一步后自动展开下一步 | 任务清单、 onboarding 步骤、表单填写 |

每种布局均内建：
- 展开/收起的箭头旋转动画（180度旋转，200ms）
- 展开内容的高度过渡动画（使用 CSS `grid-template-rows` 或 `max-height`）
- 展开状态的记忆（通过 props 控制或 localStorage 可选）
- 与 `Table`、`Item`、`Card`、`Empty` 的衔接

## 目标用户

- 需要实现主从详情、嵌套层级、分步引导的列表展示的开发者
- 希望在有限空间内展示更多层次信息的开发者

### 涉及的技术点

- 使用现有 `ui/accordion/` 或 `ui/collapsible/` 组件作为展开/收起的底层
- 使用 `ui/table/` 或 `ui/item/` 构建列表行
- 使用 CSS `transition` + `grid-template-rows: 0fr / 1fr` 实现高度动画
- 使用 `lucide-vue-next` 的 `ChevronDown` / `ChevronRight` 图标，展开时旋转 180/90 度
- 使用 `Badge` 或自定义图标展示层级状态（完成 ✓、进行中 ●、待办 ○）

### 与现有 layouts/ 的衔接

- 新增 `src/components/layouts/MasterDetailListLayout.vue`
- 新增 `src/components/layouts/NestedListLayout.vue`
- 新增 `src/components/layouts/StepExpandListLayout.vue`
- 在 `src/components/layouts/index.ts` 中导出
- `MasterDetailListLayout` 可与 `DataTableLayout` 组合（表格行展开详情）
- `NestedListLayout` 可与 `TreeSidebarLayout`（205）组合（文件树作为独立页面布局）
- `StepExpandListLayout` 可与 `FormWizardLayout`（200）组合（步骤式向导的另一种表达）

**Props 设计（以 MasterDetailListLayout 为例）：**

```ts
interface ExpandableListItem {
  id: string
  title: string
  subtitle?: string
  expanded?: boolean
  content?: unknown         // 展开后的内容数据
}

interface MasterDetailListLayoutProps {
  items: ExpandableListItem[]
  expandMode?: 'single' | 'multiple'  // 单开/多开，默认 multiple
  showExpandIcon?: boolean
  expandOnRowClick?: boolean // 点击整行展开，默认 true
}

// Events
// @expand   — 展开某行
// @collapse — 收起某行
```

## 验收标准

- [ ] 三种布局均能在 `ComponentPlayground` 中独立预览
- [ ] 展开/收起的箭头图标有旋转动画（200ms，ease-out）
- [ ] 展开内容的高度变化有过渡动画，无突兀跳动
- [ ] `expandMode="single"` 时，展开新行自动收起其他已展开行
- [ ] `StepExpandListLayout` 的步骤完成后自动展开下一步，已完成步骤可收起
- [ ] 嵌套列表支持无限层级，每层缩进 24px（或自定义）
- [ ] 空列表时展示 `Empty` 占位
- [ ] 暗色模式下展开内容的背景色与主行有微妙区分

## 优先级

P1 — 可展开列表是信息密集型桌面应用的核心布局模式。
