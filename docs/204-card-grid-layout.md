# 204 卡片网格响应式布局（Card Grid Layout）

## 功能名称

卡片网格响应式布局 — 提供响应式列数、列表/网格切换视图、筛选工具栏的卡片展示骨架。

## 功能背景/动机

卡片网格是展示资源库、项目列表、模板市场、媒体库等内容的主流布局。现有脚手架已有 `Card` 组件族和 `DashboardLayout`（内含简单的统计卡片网格），但缺少**通用的、可复用的卡片网格页面模板**。开发者在实现卡片网格时，需要重复编写响应式断点逻辑、视图切换状态、筛选/排序工具栏和空状态处理。

本功能将这些通用模式抽象为 Layout 组件，开发者只需提供卡片数据和渲染方式，即可获得一致的卡片浏览体验。

## 功能描述

包含以下卡片网格布局变体：

| 变体 | 说明 | 适用场景 |
|------|------|----------|
| `CardGridLayout` | 标准响应式网格，支持列数配置 | 通用资源展示、项目卡片 |
| `CardGridWithToolbarLayout` | 网格 + 顶部工具栏（搜索/筛选/排序/视图切换） | 带检索功能的卡片库 |
| `CardGridMasonryLayout` | 瀑布流布局，卡片高度不等 | 图片画廊、灵感板、模板市场 |

每种布局均内建：
- 响应式断点：1 列（小于 640px）→ 2 列（640-1024px）→ 3 列（1024-1440px）→ 4 列（大于 1440px）
- 列表/网格切换（在 `CardGridWithToolbarLayout` 中）
- 选中状态管理（多选框、批量操作栏）
- 空状态（`Empty` 组件）和加载状态（`Skeleton` 占位卡片）
- 分页或无限滚动占位槽

## 目标用户

- 需要实现资源库、项目列表、模板市场、相册等卡片化展示的开发者
- 希望在仪表盘内嵌入卡片预览区的开发者

### 涉及的技术点

- 使用 CSS Grid 的 `grid-template-columns: repeat(auto-fill, minmax(280px, 1fr))` 实现响应式网格
- 列表视图切换为 `flex-col` 单列布局
- 瀑布流使用 CSS Columns 或 Grid 的 masonry 支持
- 使用现有 `Card`、`Empty`、`Skeleton`、`ToggleGroup`、`Input`、`Button` 组件
- 批量操作栏使用 `AnimatedTransition` 做出现/消失动画

### 与现有 layouts/ 的衔接

- 新增 `src/components/layouts/CardGridLayout.vue`

**Props 设计（以 CardGridWithToolbarLayout 为例）：**

```ts
interface CardGridWithToolbarLayoutProps {
  items: unknown[]
  viewMode?: 'grid' | 'list'
  columns?: {                        // 自定义响应式断点
    sm?: number
    md?: number
    lg?: number
    xl?: number
  }
  loading?: boolean
  selectedIds?: string[]
  showSearch?: boolean
  showFilter?: boolean
  showSort?: boolean
  showViewToggle?: boolean
}

// Slot 约定
// #toolbar-left   — 工具栏左侧自定义内容
// #toolbar-right  — 工具栏右侧自定义内容
// #card="{ item }" — 单卡片渲染
// #empty          — 空状态覆盖
// #skeleton       — 加载状态覆盖
// #bulk-actions   — 批量操作栏内容
```

## 验收标准

- [ ] 三种布局均能在 `ComponentPlayground` 中独立预览
- [ ] 网格列数随窗口宽度自动调整：1 → 2 → 3 → 4 列
- [ ] 列表/网格切换时，卡片布局平滑过渡
- [ ] 批量选择时，底部出现批量操作栏，取消全选后自动消失
- [ ] 加载状态时展示 6-8 个 `Skeleton` 占位卡片
- [ ] 瀑布流布局中，各列高度差控制在最小卡片的 50% 以内
- [ ] 所有工具栏控件通过事件抛出，不绑定具体业务逻辑

## 优先级

P1 — 卡片网格是内容展示型桌面应用的核心布局，对脚手架的通用性提升很大。
