# 小黑助手 - UI 组件库 任务规划

## 项目概述

本项目是基于 Electron + Vue 3 + TypeScript + Vite + shadcn-vue 的桌面应用，旨在打造一个易用、美观的 UI 组件库框架。

## 已完成工作

## 工作流程

1. 每个 Agent 在自己的分支上工作
2. 完成任务后提交 PR 到 main 分支
3. 代码审查与合并
4. 持续迭代，不断优化

---

## 注意事项

- 保持代码风格一致（遵循 ESLint + Prettier）
- 所有代码必须有 TypeScript 类型定义
- 新增功能需要有对应文档
- 保持组件的可访问性（a11y）
- 优先使用 shadcn-vue 现有的组件和样式系统
